# Máfia da Graxa - Loja (Vite + React + TypeScript)

Projeto de e-commerce fictício para a equipe *Máfia da Graxa*.
Tema: Dark Mode Industrial / Street (Preto + Laranja #FF5500)

Instruções rápidas:
1. Instale dependências: `pnpm install` (ou `npm install` / `yarn`)
2. Rode em dev: `pnpm dev`

Estrutura gerada automaticamente pelo assistente.
